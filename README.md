Project contains:  
1. Generate Sudoku generator  
2. Generate Sudoku board to image file  
  
Have fun :)  
  
Used [MNIST Database](http://yann.lecun.com/exdb/mnist/) for generating sudoku board.  
  
  

# Requirement
- Python
- Numpy
- matplotlib

# Usage
### Create New Sudoku

```
python main.py
```
